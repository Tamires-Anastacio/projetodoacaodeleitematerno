Sobre o Projeto

Nosso projeto visa conectar mães e famílias que necessitam da doação de leite materno com pessoas dispostas a realizar este ato de amor por uma vida nova. Aqui nosso objetivo alem de informar como se deve realizar a doação/recebimento em instituicões mais próximas de você ,realizar o contato com instituições parceiras para maior agilidade para este serviço tão necessário.

Funcionalidades

Solicitção de interesse para doação,

Chat direto com Instituições parceiras ou devs.

Visualização de doadoras por região (apenas instituições)

Solicitação de doação de leite
